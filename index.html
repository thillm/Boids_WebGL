Create new Boid Animations Using a Genetic Algorithm

This program will combine flocking animations that you select to create new animations.
Your goal as the animator is to create a flocking animation that best shows one of the following emotions: Anger, Fear, Joy, Sadness or Surprise.
When you finish please share your creation with us by filling out the short survey here





Instructions
Pick a status for each animation.
By default animations will take part in the combining process to create new animations,
but will not appear in the next generation.
The "Remember" button keeps the selected animation so it will appear in the next generation.
The "Remove" button will remove all trace of the selected animation when rating is finished.
Removed animations will not contribute to new animations in the next generation.
